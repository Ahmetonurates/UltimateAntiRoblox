# NoCheat

Ultimate Anti-Roblox Script

This script protects your Roblox game from cheaters.  
Features:
- Detects speed, fly, infinite jump, no-clip, god mode, teleport hacks.
- Monitors chat messages and text labels for inappropriate words.
- Automatically kicks/bans cheaters.
- Whitelist system to keep trusted players safe.
- Single script for ServerScriptService.
- Auto-updates blacklist from GitHub JSON.

**How to use:**
1. Copy the script into ServerScriptService in Roblox Studio.
2. Customize the blacklist URL if needed.
3. Publish your game and enjoy cheat-free gameplay.
